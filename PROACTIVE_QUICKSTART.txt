## ⚙️ Configuration

### **File: jarvis_config.json**
```json
{
  "autonomy_level": "medium",        # low/medium/high
  "proactivity_enabled": true,       # Toggle suggestions
  "background_loop_interval": 30,    # Seconds between cycles
  "max_prediction_confidence": 0.8,  # Min confidence threshold
  "behavior_tracking_enabled": true, # Track patterns
  "risk_assessment_enabled": true,   # Risk management
  "debug_mode": false                # Verbose logging
}
```

### **Autonomy Levels Explained**
```
LOW:     Conservative, ask for everything
         ✓ Auto-execute: low-risk only
         ? Ask: medium/high/critical
         ✗ Block: critical

MEDIUM:  Balanced (recommended for start)
         ✓ Auto-execute: low-risk, high confidence
         ? Ask: medium-risk
         ✗ Block: high/critical

HIGH:    Aggressive, auto-execute most
         ✓ Auto-execute: low/medium-risk
         ? Ask: high-risk
         ✗ Block: critical only
```

**Recommendation:** Start with MEDIUM, increase after validation
